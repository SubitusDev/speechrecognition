# speechrecognition

Programa para corroborar que los audios enviados por el locutor sean correctos.

```bash
  |_ speech.sh
  |
  |_ archivos
    |
    |
    |_ audios.csv
    |_ audio01.wav
    |_ audio02.wav
    |_ ...
    |_ audio10.wav
```

Para ejecutar es necesario ingresar el comando:

```bash
./speech.sh
```
